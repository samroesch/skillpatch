# Skillpatch Library

You have access to a curated skill library via two MCP tools:
- `browse_skills(category?)` — see what's available; pass a category to drill in
- `find_skill(id)` — load full guidance for a specific skill

## When to use

Use your judgment — the same way you'd decide whether to look something up. If you're confident you know how to do something, just do it. If you think there's probably specialized guidance worth having (a niche file format, a specific workflow, a tool you don't know well), call `browse_skills()` first.

## Categories

- **productivity** — meeting notes, internal communications
- **documents** — Word, Excel, PowerPoint, PDF
- **data** — CSV analysis, D3.js visualization
- **development** — landing pages, frontend design, web apps, MCP servers, testing

Call `browse_skills()` with no arguments to see the full index with featured and top-ranked skills.
